******** HELLO WORLD ******
1 - install nodeJS
2 - check for node version:
    node -v
3 - installation of angular/cli
    npm install -g @angular/cli@VERSION (15)

4 - check if angular is successfully installed
    ng version
5 - create a new Angular project:
    ng new app_name
6 - run the app
    ng serve

7 - create our first componenet:
    ng g c comp_name
